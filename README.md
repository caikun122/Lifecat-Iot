# STM32-Album
Lifecat系统之STM32开发--STM32读取SD卡显示JPG图片实现

### 主要功能

1.通过FatFS方式读取Fat32格式SD卡中的jpg图片数据

2.通过stm32的FSMC模拟8080时序驱动LCD显示屏显示图片

3.通过循环函数以及延时函数实现循环播放图片功能

4.通过KEY1按键复位程序
